## 功能特性

### 🎯 核心功能

- **多级别日志记录**：支持 DEBUG、INFO、WARNING、ERROR、CRITICAL 五个级别
- **多种输出方式**：控制台、文件、JSON格式输出
- **灵活格式化**：文本、JSON、彩色格式化
- **安全脱敏**：内置敏感信息识别和脱敏
- **错误处理**：全局错误处理和分类处理
- **指标收集**：日志相关的性能指标收集

### 🔧 高级特性

- **配置驱动**：YAML配置文件和环境变量支持
- **线程安全**：多线程环境下的安全日志记录
- **高性能**：缓存机制和异步处理能力
- **可扩展**：插件化的处理器和格式化器架构
- **生产就绪**：完整的错误处理和监控支持

## 组件概览

### 核心层 (src/core/logger/)

```
src/core/logger/
├── log_level.py              # 日志级别定义
├── formatters/               # 格式化器
│   ├── base_formatter.py     # 基础格式化器
│   ├── text_formatter.py     # 文本格式化器
│   ├── json_formatter.py     # JSON格式化器
│   └── color_formatter.py    # 彩色格式化器
└── handlers/                 # 处理器
    ├── base_handler.py       # 基础处理器
    ├── console_handler.py    # 控制台处理器
    ├── file_handler.py       # 文件处理器
    └── json_handler.py       # JSON处理器
```

### 服务层 (src/services/logger/)

```
src/services/logger/
├── logger.py                 # 日志记录器实现
├── redactor.py               # 日志脱敏器
├── error_handler.py          # 错误处理器
├── metrics.py                # 指标收集器
├── structured_file_logger.py # 结构化文件日志记录器
└── log_level.py              # 日志级别定义
```
